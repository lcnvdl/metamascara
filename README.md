## Installation

CDN
Import the script
```html
<script src="https://unpkg.com/mmascara/lib/metamascara.min.js"></script>
```

Then
```javascript
const { MetaMascaraFactory } = mmascara;
```

ANGULAR
Node
```bash
npm i --save mmascara
```

Then
```typescript
import { MetaMascaraFactory } from 'mmascara';
```
